# Rslib project

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Build the library:

```bash
pnpm build
```

Build the library in watch mode:

```bash
pnpm dev
```

## Benchmarking

Run the local core benchmarks:

```bash
pnpm bench
```

The benchmark suite uses Vitest benchmark mode and prints comparison tables to
the console only. It does not write result artifacts to disk.
